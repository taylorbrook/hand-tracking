Max/MSP Integration Instructions

Method 1: Using Node for Max

  1. Install Node for Max from the package manager
  2. Use [node.script] object to run a local server
  3. Send OSC data from this page to Max via UDP

Method 2: WebSocket (Recommended)

  1. Run this HTML file locally or on a server
  2. In Max, use [udpreceive] or websocket objects to receive data
  3. Parse incoming landmark data (21 landmarks per hand, each with x, y, z)

Landmark Indices:

  • 0: Wrist
  • 1-4: Thumb (CMC, MCP, IP, Tip)
  • 5-8: Index Finger
  • 9-12: Middle Finger
  • 13-16: Ring Finger
  • 17-20: Pinky
